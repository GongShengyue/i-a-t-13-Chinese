# i-a-t-13-Chinese
The simplify Chinese translation of i-a-t-13 MSFS addon
I tanslated the addon by using Tencent TMT,(腾讯的机器翻译API)所以翻译的可能有些拗口。
Usage:
  unzip the Chinese.7z file and copy the 4 zh-CN.locPak files to your mod files.
